# Звуки

В этой игре звуки **не хранятся в виде файлов**, а синтезируются на лету через
Web Audio API (см. `src/core/AudioManager.js`).

Почему так:
- нулевой вес и мгновенная загрузка — важно для требования «старт < 3 секунд»;
- ничего не нужно качать по сети, игра полностью работает офлайн;
- это и есть «аналог Howler.js», построенный на нижележащем Web Audio.

Синтезируются: раздача карты, переворот, клик кнопки, фишка, победа,
блэкджек, проигрыш, перебор и ничья.

Если захотите заменить на реальные аудиофайлы — положите их сюда и подключите
загрузку в `AudioManager.js` (метод `play` уже централизован).
