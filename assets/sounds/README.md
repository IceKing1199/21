# Звуки

В этой игре звуки **не хранятся в виде файлов**, а синтезируются на лету через
Web Audio API (см. `src/core/AudioManager.js`).

Почему так:
- нулевой вес и мгновенная загрузка — важно для требования «старт < 3 секунд»;
- ничего не нужно качать по сети, игра полностью работает офлайн;
- это и есть «аналог Howler.js», построенный на нижележащем Web Audio.

Из файлов проигрываются: **перемешивание** (`shuffle.wav`) и **раздача карты**
(`deal.wav`) — они загружаются и декодируются на экране загрузки и воспроизводятся
через тот же Web Audio (`AudioManager.playSample`).

Синтезируются по-прежнему: переворот, клик кнопки, фишка, победа, блэкджек,
проигрыш, перебор и ничья.

Если захотите заменить ещё какие-то звуки на файлы — положите их сюда и добавьте
имя→URL в `SAMPLE_URLS` внутри `AudioManager.js` (метод `play` уже централизован
и сам выберет файл, а при его отсутствии откатится на синтез).
